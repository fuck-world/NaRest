为什么要这个框架?

1 现在大部分框架都很沉重,建一个项目需要依赖太多无用的包

2 针对自己目前的需求而开发

有什么好处？

1 可以一套核心共用多个项目，互相不排斥

2 独立项目之间可自行建立不同需求扩展，组件，包，等

全局使用命名空间，支持PHP PSR4 PSR0 规范

Config 配置解释
